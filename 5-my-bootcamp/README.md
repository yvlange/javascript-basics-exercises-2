# Exercise 5: My Bootcamp

Create an Array of strings containing all the people in your bootcamp class.

Display to the console (using `console.log`) each name prefixed with the order of
appearance in the array

## Example

- `['Marwin' , 'Namir']`
- Output:

```js
"#1 Marwin";
"#2 Namir";

```
